## Dependencies
Project is created with:
* React
* React Router 
* Use Clipboard Copy

## Steps on how to clone
#### 1: Clone this repo using this command 
`git clone https://github.com/NaNa2005/MemeMaker-with-React.js`

#### 2: Enter 
`cd MemeMaker-with-React.js`

#### 3: Next
`npm install`

#### 4: Enter 
`yarn start` or `npm start`
